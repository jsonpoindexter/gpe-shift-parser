# gpe-shift-parser
Parse shifts from CSV and put them in a google sheets document.

# Requirements
* Python 2
* `pip install urllib2`
* [Google Cloud Credentials](https://console.cloud.google.com/) (for editing gdrive/gsheets)
* [Babalooey](https://www.babalooey.com/) Admin Credentials
: 

# Run
`python main.py`
